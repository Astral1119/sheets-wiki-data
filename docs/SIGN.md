Given an input number, returns `-1` if it is negative, `1` if positive, and `0` if it is zero.

### Sample Usage

```gse
SIGN(-42)
SIGN(A2)
```

### Syntax

```gse
SIGN(value)
```

- `value` - The value whose sign will be evaluated.
