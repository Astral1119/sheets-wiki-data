The SECH function returns the hyperbolic secant of an angle.

### Syntax
```gse
SECH(value)
```

| Part | Description |
| --- | --- |
| `value` | Any real value to calculate the hyperbolic secant of. |

### Sample formulas

```gse
SECH(A1)
SECH(1)
```
