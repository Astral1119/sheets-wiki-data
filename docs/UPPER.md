Converts a specified string to uppercase.

### Sample Usage

`UPPER("lorem ipsum")`

`UPPER(A2)`

### Syntax

`UPPER(text)`

* `text` - The string to convert to uppercase.

### See Also

[[PROPER]]: Capitalizes each word in a specified string.

[[LOWER]]: Converts a specified string to lowercase.
