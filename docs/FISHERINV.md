Returns the inverse Fisher transformation of a specified value.

### Sample Usage

`FISHERINV(0.962)`

`FISHERINV(A2)`

### Syntax

`FISHERINV(value)`

* `value` - The value for which to calculate the inverse Fisher transformation.
