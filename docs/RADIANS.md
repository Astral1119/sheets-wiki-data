The RADIANS function converts an angle value in degrees to radians.

### Sample Usage

```gse
RADIANS(180)
RADIANS(6)
RADIANS(A2)
```

### Syntax

```gse
RADIANS(angle)
```

- `angle` - The angle to convert from degrees to radians.
