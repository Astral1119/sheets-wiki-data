Converts a specified string to lowercase.

### Sample Usage

```gse
LOWER("LOREM IPSUM")
LOWER(A2)
```

### Syntax

```gse
LOWER(text)
```

- `text` - The string to convert to lowercase.

[[UPPER]]: Converts a specified string to uppercase.
