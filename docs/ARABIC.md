Computes the value of a Roman numeral.

### Sample Usage

```gse
ARABIC("XIV")
ARABIC("MMXIII")
```

### Syntax

```gse
ARABIC(roman_numeral)
```

- `roman_numeral` - The Roman numeral to format, whose value must be between 1 and 3999, inclusive.
